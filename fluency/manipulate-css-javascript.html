Manipulating CSS Class Properties Using JavaScript

This example demonstrates how you can trigger a Javascript function by clicking in the menu icon, display the menu options and change the color of the menu manipulating the CSS with Javascript code


function manipulateCSS() { var nav = document.getElementById("nav"); var dropDown = document.getElementById("myLinks"); if (dropDown.style.display === "block") { dropDown.style.display = "none"; nav.style.backgroundColor = "#333"; } else { dropDown.style.display = "block"; nav.style.backgroundColor = "green"; } }
